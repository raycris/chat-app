# Chat app
This chat app it's like a private chat with another person who's need create a account to participate in the chat. 


## Overview

This site using the following stack:

- Framework - React
- Language - TypeScript
- Styling - Tailwind CSS
- Linting - ESLint
- Formatting - Prettier
- API - Stream https://getstream.io/
